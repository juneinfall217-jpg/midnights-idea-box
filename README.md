# Midnights

一个轻量级个人灵感管理网站。当前版本是纯静态网页，可以直接部署到 Netlify、Vercel 或 GitHub Pages。

## 使用方式

- 打开 `index.html` 就能使用。
- 新建、编辑、删除的灵感会保存在当前浏览器的本地存储里。
- 不需要账号、不需要服务器。

## 发布给别人使用

### Netlify

1. 打开 Netlify 的部署页面。
2. 将整个 `idea-box` 文件夹拖进去。
3. Netlify 会生成一个公开链接，别人打开链接即可使用。

### Vercel

1. 新建一个静态项目。
2. 上传或连接包含 `index.html` 的文件夹。
3. 不需要设置 build command。

### GitHub Pages

1. 新建一个仓库。
2. 上传 `index.html` 和 `assets` 文件夹。
3. 在仓库设置里开启 Pages。

## 重要说明

当前版本的数据是“每个人存在自己的浏览器里”。也就是说：

- 朋友打开链接后，可以创建自己的灵感库。
- 你们之间的数据不会互相看见。
- 换浏览器、清缓存或换设备，原来的本地数据可能不会自动同步。

如果以后需要账号登录、多人共享、跨设备同步，就需要接入数据库和后端服务。
